Uizard

Sketch-Based Image Retrieval

PM: Luke Dai

Project Description

Sometimes finding images of what you want is difficult using language (for example, you may not know the name of a certain thing you are looking for). An interesting approach that can help in some cases is to make a quick freehand sketch and search for similar images to the sketch. This project will dive deep into computer vision and deep learning, including topics such as Image Processing, Convolutional Neural Networks, and Generative Adversarial Networks. We will be working with Uizard, a rapidly growing startup that is seeking to revolutionize web and user interface design.
